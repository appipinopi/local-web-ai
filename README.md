*Unrestricted AI (Browser Edition) - Ver.0.0.5*

Unrestricted AI は、ブラウザひとつで完結する、制限のない学習型テキスト生成AIです。
サーバーやPython環境は不要。単一のHTMLファイルのみで動作し、入力されたあらゆるテキストデータを無差別に学習・模倣します。

✨ 主な機能 (Features)

完全ローカル動作: 外部サーバー（AIプロバイダ）にデータを送信しません。あなたのPC内で完結します。

3つの学習ソース:

Web Crawler: 指定したURLを巡回し、サイト内のテキストを自動収集。

Text Injection: テキストの貼り付け、または .txt ファイルの読み込み。

Chat Learning: 会話を通じたリアルタイム学習。

データ合成 (Synthesis): 複数の学習データ（JSON）を合体させ、異なる知識を持つ「キメラAI」を作成可能。

カンマ区切りモード: 単語リストを一括で学習させるための専用モード搭載。

揮発性メモリ: プライバシー保護のため、ブラウザを閉じると学習データは消去されます（手動保存が可能）。

🚀 使い方 (Usage)

ai_v005.html をダウンロードします。

PCのブラウザ（Chrome, Edge, Firefoxなど）でファイルを開きます。

起動時に表示される警告画面の内容を確認し、「同意して起動」をクリックします。

📚 学習方法 (How to Train)

1. Web Crawler (URL学習)

Webサイトの知識を取り込みます。

URLを入力して「開始」を押すと、リンクを辿りながら（最大30ページ）テキストを収集します。

Note: allorigins.win プロキシを経由してデータ取得を行います。

2. Text Injection (テキスト注入)

手持ちのテキストデータを与えます。

Manual Input: テキストエリアに文章を貼り付けて「学習」ボタンを押します。

Comma Mode: チェックを入れると、カンマ(,)で区切られた単語をそれぞれ独立したデータとして学習します（単語帳の学習に最適）。

.txt 読込: ローカルにあるテキストファイルを読み込ませます。

3. Chat Interface (対話学習)

会話学習モード: チェックを入れると、あなたが送信したメッセージを即座に学習データに追加します。口癖や性格をリアルタイムで教え込むことができます。

💾 データ管理 (Memory Management)

保存 (Save): 現在の「脳」の状態を .json ファイルとしてダウンロードします。

上書き読込: 現在の記憶を消去し、ロードしたデータで置き換えます。

＋合成読込 (Merge): 現在の記憶を保持したまま、ロードしたデータを追加・統合します。

🛠 技術仕様 (Technical Specs)

Core: N-gram Markov Chain (N=3)

Stack: HTML5, CSS3, Vanilla JavaScript

Storage: Local JSON File (No Database required)
